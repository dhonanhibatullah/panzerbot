# Project Description
* This Next.js frontend project will be deployed on the same machine as the backend.
* It serves a single-page app that displays the robot's camera view and receives the robot's microphone via WebRTC.
* It also captures the client's microphone and sends it to the robots via WebRTC.
* The RTC signalling is handled by the backend.
* It controls the robot via WebSocket.
